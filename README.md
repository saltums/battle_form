# 🎸 バンド陣形告知ジェネレーター

レトロRPG（ロマサガ風）の戦闘画面をモチーフにした、バンドのライブ告知用画像ジェネレーターです。
ブラウザだけで動作し、サーバーやデータベース（GAS等）を必要としないスタンドアロンなツールです。

## ✨ 主な機能
- **陣形システム**: インペリアルクロス、トライアングル、ダイヤモンドなど、人数に合わせた配置をワンクリックで切り替え。
- **ドラッグ＆ドロップ**: メンバーのアイコンをマウスやタッチ操作で自由に微調整可能。
- **カスタマイズ**: メンバー名、パート、アイコン画像URL、セリフ、背景デザインを自由に変更できます。
- **SNS投稿最適化**: 「SNS用画像を保存する」ボタンで、高画質なPNG画像をダウンロード可能。
- **ローカル保存**: 入力した内容はブラウザ（localStorage）に自動保存されるため、次回起動時もそのまま再開できます。

## 🚀 使い方
1. `index.html` をブラウザで開きます。
2. 告知内容（会場、日時、ライブ名）を入力します。
3. 陣形タイプを選択するか、メンバーアイコンをドラッグして配置を決めます。
4. 「SNS用画像を保存する」ボタンを押して画像を保存します。
5. （高度な設定）「メンバー情報の詳細設定」から、各メンバーの名前やアイコン画像をURL指定で変更できます。

## 🛠 技術構成
- HTML5 / CSS3 / JavaScript (Vanilla)
- [html2canvas](https://html2canvas.hertzen.com/) (画像生成ライブラリ)
- localStorage (データ永続化)

## 📦 インストール / 公開方法
このリポジトリをGitHubにアップロードし、GitHub Pagesを有効にするだけで公開可能です。
ビルド手順やバックエンドの設定は一切不要です。
